
# 🎨 Modern Background Color Changer

A modern and responsive Background Color Changer built using **HTML, CSS, and JavaScript**.
This project allows users to change the webpage background with different colors and beautiful linear gradients.

## ✨ Features

* 🎨 Multiple background color options
* 🌈 Linear gradient backgrounds
* 🔄 Reset button
* 📱 Responsive design
* ✨ Smooth hover effects
* 🕺 Animated heading
* ⚡ Interactive UI using JavaScript DOM manipulation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
